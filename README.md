# Wapp

Multi-account WhatsApp Web client for Mac.

## Features

- **Multiple accounts** — each account runs in its own isolated session with separate QR login
- **Toggleable sidebar** — hide the chat list to focus on a conversation (`⌘B`)
- **Tab position** — switch between top and left tab layouts via settings
- **Right-click menu** — rename, reload, or remove accounts
- **Persistent sessions** — stay logged in between app restarts

## Install

```bash
bun install
```

## Run

```bash
bun run start
```

## Build

```bash
bun run build
```

## Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘B` | Toggle WhatsApp sidebar |
| `+` button | Add new account |
| Right-click tab | Rename / Reload / Remove |
| Settings icon | Switch tab position (top/left) |
